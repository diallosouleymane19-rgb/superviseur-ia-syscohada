\## 🎯 Présentation



Le \*\*Superviseur IA Comptable SYSCOHADA\*\* est une application web intelligente

spécialement conçue pour les entreprises et cabinets comptables des pays membres

de l'\*\*UEMOA\*\* et de la zone \*\*OHADA\*\*.



Propulsé par l'IA Mistral, il automatise la production et l'analyse des états

financiers selon les normes \*\*SYSCOHADA révisé\*\*.

\## ✨ Fonctionnalités



| Module | Description |

|--------|-------------|

| 🏢 \*\*Dossiers Entreprises\*\* | Gestion complète des dossiers avec historique des analyses |

| 📊 \*\*Analyse Balance\*\* | Analyse IA selon normes SYSCOHADA avec SIG et ratios |

| 📋 \*\*Bilan SYSCOHADA\*\* | Génération automatique du bilan selon modèle OHADA |

| 📈 \*\*Compte de Résultat\*\* | SIG, marges, RAO et résultat net en FCFA |

| 💰 \*\*TAFIRE\*\* | Tableau Financier des Ressources et Emplois |

| 📝 \*\*Notes Annexes\*\* | Génération automatique des notes annexes obligatoires |

| 🧾 \*\*Liasse Fiscale\*\* | Liasse fiscale adaptée à chaque pays UEMOA |

| 🔍 \*\*Plan Comptable OHADA\*\* | Consultation et recherche dans le plan SYSCOHADA |

| 📰 \*\*Veille Fiscale UEMOA\*\* | Actualités fiscales par pays |

| 📥 \*\*Export HTML + Word\*\* | Rapports professionnels téléchargeables |

\## 📦 Installation locale



```bash

\# 1. Cloner le dépôt

git clone https://github.com/diallosouleymane19-rgb/superviseur-ia-syscohada

cd superviseur-ia-syscohada



\# 2. Installer les dépendances

pip install -r requirements.txt



\# 3. Configurer la clé API Mistral

\# Créer un fichier .env à la racine :

MISTRAL\_API\_KEY=votre\_clé\_api



\# 4. Lancer l'application

streamlit run app.py

```
